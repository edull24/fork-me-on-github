Fork Me On GitHub

A Simple CSS "Fork Me On GitHub" Banner

So yeah, that dude to the right is pretty much it. Nothing revolutionary, but if you want a dynamic, customizable, and easy to use banner without an extra http request then have at it.

Features

  • Super simple markup: 
    <div class="fork-me right">
    <a href="[your repo url]">Fork Me On GitHub</a>
</div>
  • Easy to override styles
  • Fully customizable with SASS support
  • Supports left or right banner, just add the respective class

Installation

  • Install with Bower: 
    bower install fork-me-on-github --save
  • Clone the git repo: 
    git clone https://github.com/edull24/fork-me-on-github.git
    • checkout the tagged release you want to use
  • Download the latest release source

Usage

Basic:

  1. Pull in the styles: 
    <link rel="stylesheet" href="[path to fork.css]">
  2. Add the markup: 
    <div class="fork-me right">
    <a href="[your repo url]">Fork Me On GitHub</a>
</div>
  3. Override the default styles by adding your own rules to your CSS file (optional)
